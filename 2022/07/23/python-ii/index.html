    Python II

    Age 8 to 14, $60

    Requirements:
     None!

    People of Programming is offering you the perfect opportunity to become a skilled programmer! Come join us as we learn about the fundamental aspects of Python.

    Note: If you would like to attend specific classes, the price is 25$/class.

    Week 1: Expressions, Data Types Concatenation and Replication, Variables, Comments, Print, Input
    Week 2: Length, Variable type functions, Comparison Operators, Boolean Operators, Time and Date
    Week 3: Conditionals, Loops, Flow Control, Functions
    Week 4: Projects Introduction
    Week 5: Working on Projects
    Week 6: Project Presentation

    Each topic will be covered extensively. By the end of the class, the student will have a good understanding of the Python programming language.
